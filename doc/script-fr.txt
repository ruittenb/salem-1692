
A. La nuit tombe. Les citoyens de Salem vont se coucher : fermez tous les yeux.

B. La nuit touche à sa fin. Les citoyens de Salem dorment encore : fermez tous les yeux.


C. Une fois que toutes les paupières sont closes, la Sorcière se réveille.

E. Sorcière, ouvrez les yeux et désignez sans faire de bruit la personne qui recevra le Chat noir.
   Il peut s'agir de vous-même.

H. Sorcière, vous avez fait votre choix : refermez les yeux.


D. Une fois que toutes les paupières sont closes, les Sorcières se réveillent.
   Sorcières, ouvrez les yeux et découvrez qui est membre de votre assemblée.

F. En communiquant par gestes, désignez sans faire de bruit la personne qui recevra le Chat noir.
   Il peut s'agir de l'une d'entre vous.

G. En communiquant par gestes, désignez sans faire de bruit la personne qui sera assassinée.
   Il peut s'agir de l'une d'entre vous.

J. Sorcières, vous avez fait votre choix : refermez les yeux.


M. Dès que les Sorcières sont endormies, le Prévôt se réveille.
   Prévôt, décidez qui sera protégé cette nuit.

P. Il ne peut pas s'agir de vous-même.

Q. Il peut s'agir de vous-même.

N. Prévôt, vous avez fait votre choix : refermez les yeux.


Z. L'aube se lève et le village se réveille : ouvrez tous les yeux.
